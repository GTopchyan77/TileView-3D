# TileView 3D

TileView 3D is a demo-ready web app for previewing floor and wall tiles inside interactive 3D room templates. It is designed as an MVP for tile shops, interior sales demos, and portfolio presentation work.

## Features

- 3D room visualizer
- Floor and wall tile selection
- Per-surface tile application
- Automatic room objects by room type
- GLB object loading
- Object move / rotate / scale controls
- Admin tile import
- Mobile-friendly layout

## Tech Stack

- Next.js
- TypeScript
- React
- React Three Fiber / Three.js
- Tailwind CSS

## How To Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Status

Demo / MVP

## Known Limitations

- No backend yet
- No real AI reconstruction from photos
- GLB quality depends on the uploaded model assets

## Future Improvements

- Backend tile catalog and admin persistence
- Saved projects and shareable customer links
- AR / mobile viewer support
- AI-assisted floor plan recognition
