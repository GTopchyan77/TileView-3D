# Agents

## Recommended Roles
- Product/PM agent: keep scope aligned to showroom MVP use cases
- Frontend agent: maintain Next.js UI and app-router structure
- 3D agent: improve scene realism, lighting, camera logic, and export
- Data/content agent: expand tile catalog JSON and texture assets

## Working Agreements
- Keep features incremental and demoable
- Preserve local-first behavior until backend work starts
- Prefer simple room templates over high-complexity modeling
- Keep texture scaling logic explicit and easy to test
