# Tile Visualizer Project Brief

## Purpose
Tile Visualizer is a lightweight showroom MVP that helps tile shops present tile finishes inside simple 3D room templates. The goal is to make product comparison faster during in-store or remote sales conversations without requiring a backend, authentication, or advanced rendering features.

## MVP Outcome
- Render 3 room templates: Bathroom, Kitchen, Living Room
- Load a local tile catalog from JSON
- Let a user apply a selected tile to floor or wall
- Approximate texture scale using tile width and height in centimeters
- Export a screenshot from the 3D viewer
- Provide a simple admin mock page to add tiles using browser-local state

## Non-Goals
- No login or permissions
- No payments, cart, or checkout
- No backend or database
- No AI texture generation
- No advanced CAD-grade room editing
