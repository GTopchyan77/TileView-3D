# Demo Script

## 2-Minute Walkthrough

1. Open the app homepage and introduce TileView 3D as a demo-ready 3D tile room visualizer.
2. Select a room type such as Bathroom, Kitchen, Living Room, Bedroom, or Empty Room.
3. Choose the exact target surface:
   - Floor
   - Left Wall
   - Right Wall
   - Back Wall
   - All Walls
4. Pick a tile from the Demo Catalog and show the room update instantly.
5. Point out the automatic room objects that load for the selected room type.
6. Select an object and adjust:
   - X position
   - Y position
   - Z position
   - Rotation
   - Scale
7. Add another object manually if needed.
8. Use Zoom In, Zoom Out, and Reset View to present the room clearly.
9. Export the preview image to show the shareable output direction for production.
