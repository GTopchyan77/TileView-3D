# Roadmap

## Phase 1: Current MVP

- Demo-ready 3D Tile Room Visualizer
- Room templates for bathroom, kitchen, living room, bedroom, and empty room
- Floor and per-wall tile application
- Automatic room object presets by room type
- Manual object placement with move, rotate, scale, and delete controls
- GLB model loading with placeholder fallback
- Admin tile import using local state and browser storage
- Mobile-friendly responsive demo flow

## Phase 2: Backend, Catalog, and Share Links

- Persistent backend tile catalog
- Saved customer projects and shareable preview links
- Admin dashboard backed by real storage
- Multi-user access and quote workflow
- Asset management for tile thumbnails and object libraries

## Phase 3: AI Floor Plan / Photo Reconstruction

- AI-assisted floor plan recognition
- Photo-to-room geometry estimation
- Smarter room layout generation from sketches or plans
- Automatic object suggestions and placement assistance
- Higher fidelity reconstruction and export workflows
